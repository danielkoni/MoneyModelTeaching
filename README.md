# Money Model Teaching

This lesson uses the Python Mesa's documentation tutorial by the Mesa Team of the Boltzmann wealth money model to show students how to code an Agent Based Model. The code has been cut up into sections in the form of simple code blocks. 

We'll start by introducing and explaining the initial model code in money_model.py and then running it in run_money_model.ipynb

Then to gradually expand the model copy paste blocks from expansion_blocks.ipynb into the model or run files to build up and demonstrate the different features of a mesa abm model.
